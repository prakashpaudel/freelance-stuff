ABOUT SANS
==========

Sans, an easy to use responsive premium WordPress portfolio theme best suited for creatives. The theme contains one custom post type and supports 6 WordPress Post Formats, including link, audio, video, gallery, image, and status.

Please read the User Guide for installation and usage instructions.

Support for Sans is available at http://support.designcrumbs.com.
